* ❌ Wrong: `para1.parentElement()` → `parentElement` is a **property**, not a function.

```js
console.log(para1.parentElement);  
```

### ✅ 2. `children`

* ❌ Wrong: `arti1.children()` → again, `children` is a **property**, not a function.
* ✅ Correct:

```js
let coll = arti1.children;   // gives HTMLCollection of all child elements of #arti

}
```
